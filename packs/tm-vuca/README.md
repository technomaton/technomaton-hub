# tm-vuca — VUCA Skills Framework

Multi-agent system for auditing and improving communication, skills, and processes across four VUCA dimensions.

## What's inside

| Type | Count | Description |
|------|-------|-------------|
| Skills | 1 | vuca-assessment (orchestration + knowledge base) |
| Agents | 5 | conductor + 4 dimension specialists |
| Commands | 3 | /vuca/audit, /vuca/redesign, /vuca/portfolio |

## Four Dimensions

| Dimension | Agent | What it evaluates |
|-----------|-------|-------------------|
| Kolaborace | @vuca-collaboration | Audience adaptation, uncertainty handling, communication |
| Perspektivy | @vuca-perspectives | Multi-source usage, cross-validation, conflict resolution |
| Kontext | @vuca-context | Situational awareness, broader contexts, constraints |
| Rozhodování | @vuca-decision | Framing, success criteria, alternatives, verification |

## Scoring

Each dimension scored 0-5, total 0-20. Target: ≥ 14/20, no dimension below 2.
