# tm-strategy — Integrated Strategic Assessment

Meta-plugin that orchestrates VUCA Skills + AI PMF Playbook for comprehensive strategic evaluation.

## What's inside

| Type | Count | Description |
|------|-------|-------------|
| Skills | 1 | strategy-integrated (VUCA + PMF orchestration) |
| Agents | 1 | strategy-conductor (meta-orchestrator) |
| Commands | 2 | /strategy:audit, /strategy:compass |

## Dependencies

- `tm-vuca` — VUCA Skills Framework (Dawson/Lectica)
- `tm-pmf` — AI PMF Playbook (Jaffer)

## Cross-Framework Integration Matrix

| VUCA Dimension | PMF Framework | Risk When Both Low |
|---|---|---|
| Contextual Thinking | Market analysis, Competition, AI Strategic Lens | Blind spots |
| Decision-Making Process | 4D Method, Launch Canvas, AI Feasibility Checklist | Premature scaling |
| Perspective Coordination | User research, Metrics, 5-Question Pain Analysis | False PMF |
| Collaborative Capacity | Trust engineering, UX, Trust Layer, Psychological Triggers | Scaling churn |

## Usage

- `/strategy:audit [target]` — Full combined VUCA + PMF assessment
- `/strategy:compass [target]` — Quick strategic positioning with 2x2 matrix
