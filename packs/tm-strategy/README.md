# tm-strategy — Integrated Strategic Assessment

Meta-plugin that orchestrates VUCA Skills (HOW) + AI PMF Playbook (WHAT) + Wardley Mapping (WHERE) for comprehensive strategic evaluation.

## What's inside

| Type | Count | Description |
|------|-------|-------------|
| Skills | 1 | strategy-integrated (VUCA + PMF + Wardley orchestration) |
| Agents | 1 | strategy-conductor (meta-orchestrator, dispatches to 3 conductors) |
| Commands | 2 | /strategy:audit, /strategy:compass |

## Dependencies

- `tm-vuca` — VUCA Skills Framework (Dawson/Lectica)
- `tm-pmf` — AI PMF Playbook (Jaffer)
- `tm-wardley` — Wardley Mapping (Simon Wardley)

## Three-Framework Integration Matrix

| VUCA Dimension | PMF Framework | Wardley (WHERE) | Risk When All Low |
|---|---|---|---|
| Contextual Thinking | Market analysis, Competition, AI Strategic Lens | Value chain visibility, climatic patterns | Strategic blind spot |
| Decision-Making Process | 4D Method, Launch Canvas, Feasibility Checklist | Build-vs-buy, evolution stage, inertia | Premature/misdirected scaling |
| Perspective Coordination | User research, Metrics, 5-Question Pain Analysis | Stakeholder positioning, ecosystem mapping | False PMF |
| Collaborative Capacity | Trust engineering, UX, Trust Layer, Psych Triggers | Doctrine adherence, strategic play consensus | Scaling churn |

## Usage

- `/strategy:audit [target]` — Full combined VUCA + PMF + Wardley assessment
- `/strategy:compass [target]` — Quick strategic positioning
