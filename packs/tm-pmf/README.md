# tm-pmf — AI Product-Market Fit Playbook

Strategic assessment plugin for AI products. Multi-agent system covering opportunity scoring, product development, moat construction, metrics, and launch readiness.

## What's inside

| Type | Count | Description |
|------|-------|-------------|
| Skills | 4 | pmf-assessment, pmf-opportunity, pmf-product-design, pmf-moat-analysis |
| Agents | 5 | conductor + 4 specialists |
| Commands | 6 | /pmf:score, /pmf:audit, /pmf:moat, /pmf:launch-check, /pmf:invisible-pain, /pmf:validate |
| Knowledge | 6 | 5 framework files + case studies |

## Key Frameworks

| Framework | Knowledge File | What it does |
|-----------|---------------|-------------|
| Pain x Frequency x AI Advantage | AI_PMF_CORE.md | Score opportunity viability in 3 zones |
| 4D Method | AI_PMF_PRODUCT.md | Discover, Design, Develop, Deploy stage gates |
| Three-Moat Taxonomy | AI_PMF_MOATS.md | Data, Behavioral, Workflow moat analysis |
| Dual Success Metrics | AI_PMF_METRICS.md | Traditional + AI-specific measurement |
| Launch Strategy Canvas | AI_PMF_LAUNCH.md | 4-dimension readiness scoring |
| Case Studies | CASE_STUDIES.md | 10 AI company analyses (Klarna, Cursor, etc.) |
