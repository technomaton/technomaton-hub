# Launch Strategy Canvas

Run a Launch Strategy Canvas assessment for the product specified in $ARGUMENTS.

Read `skills/pmf-assessment/AI_PMF_LAUNCH.md` from this pack for the full framework.

Score all 4 dimensions on a Green / Yellow / Red scale:
- **Customer Readiness** — Do users pull the product or do you push it?
- **Product Readiness** — Is reasoning stable, are edge cases handled?
- **Company Readiness** — Team, funding, support capacity in place?
- **Competition Readiness** — Is the window open, moats defensible?

For each dimension, provide specific evidence and rationale for the color rating.

Run the **Pre-Launch Checklist** from the framework and flag any blockers.

Output the full canvas with a clear **Go / No-Go recommendation**.

Key principle: "You only scale when all four are green."

Default output language is Czech. If the user writes in English, respond in English.
