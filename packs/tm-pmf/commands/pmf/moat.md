# Moat Analysis

Analyze competitive moats for the product specified in $ARGUMENTS.

Invoke @pmf-moat to perform a full moat assessment:

1. **Score all 5 moat types** (each 1-10):
   - Data Moat — proprietary data assets and feedback loops
   - Behavioral Moat — user habits, switching costs, workflow lock-in
   - Workflow Moat — deep integration into existing processes
   - Distribution Moat — channel advantages and network effects
   - Trust Moat — brand authority, compliance, certifications

2. **Assess clone risk** — how quickly could a competitor replicate the core value?

3. **Identify commoditization timeline** — when will the AI capability become table stakes?

4. **Recommend moat construction priorities** — which moats to build first and how.

5. **Include comparison to relevant case studies** from the framework where applicable.

Default output language is Czech. If the user writes in English, respond in English.
